Hexo 发布文章四步走：创建 post，本地运行预览，生成静态页面，部署到 github。

一、发布文章

1.创建新的 post

1

hexo new [post] "My New Post Title"

详见官网： Writing

2.本地服务器运行预览

1

hexo server

详见官网： Server

3.生成静态文件

1

hexo generate

详见官网： Generating

4.部署到网站

1

hexo deploy

生成后自动部署（以下命令是等价的）

1
2

hexo g -d
hexo d -g

详见官网： Deployment
